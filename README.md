# bachelor-landing-crm

Landing page for bachelor's thesis about a CRM system for an electronics repair workshop.

## Development strategy
This project follows **Trunk-Based Development**.  
All changes are integrated into the `main` branch through small, meaningful commits.

## Project structure
- `index.html` — main landing page
- `style.css` — styles and responsive layout
- `robots.txt` — search engine rules
- `sitemap.xml` — sitemap
- `favicon.svg` — site icon
- `images/` — visual assets
