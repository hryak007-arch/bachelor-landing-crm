Бакалаврська робота / Bachelor’s Thesis

CRM-система для майстерні ремонту електроніки

CRM System for an Electronics Repair Workshop

Веб-рішення для обліку клієнтів, обробки заявок, контролю статусів ремонту та формування звітності у невеликій сервісній майстерні.

Репозиторій Контакти

Актуальність теми

Малим майстерням часто бракує зручного цифрового інструмента для централізованого ведення клієнтських заявок, контролю етапів ремонту та швидкого формування звітності. Запропонована CRM-система зменшує ручну роботу та підвищує прозорість процесів.

Короткий опис: Односторінкове представлення бакалаврської роботи про створення CRM-системи на FastAPI та SQLite для автоматизації роботи сервісної майстерні.

Ключові слова: CRM, FastAPI, SQLite, сервісна майстерня, електроніка, веб-розробка, автоматизація.

Мета дослідження

Розробити та обґрунтувати веб-CRM систему для майстерні ремонту електроніки, яка забезпечить ефективний облік клієнтів, заявок, статусів виконання та звітних даних.

Основні завдання

  1. Проаналізувати потреби майстерні та визначити ключові бізнес-процеси.
  2. Спроєктувати структуру даних для клієнтів, заявок і статусів обслуговування.
  3. Реалізувати серверну частину системи на FastAPI із використанням SQLite.
  4. Створити зручний інтерфейс для перегляду, додавання та оновлення заявок.
  5. Передбачити базову звітність для контролю роботи майстерні.

Методологія дослідження

У роботі використано підхід прикладного проєктування веб-системи: аналіз предметної області, моделювання структури даних, реалізація серверної логіки, тестування та оцінка придатності рішення для невеликого бізнесу.

Очікувані результати

  • Прототип CRM-системи для реєстрації клієнтів і заявок.
  • Можливість змінювати статуси виконання ремонту.
  • Формування базової звітності щодо звернень.
  • Спрощення внутрішнього обліку та підвищення керованості процесу.
